My portfolio site that showes my products built with Next.js, Three.js and R3F.

## Getting Started

Install dependencies:
```bash
npm install
# or
yarn install
```

Start the project:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
